# Portrait / Headshot

Replace `jimmy-headshot.jpg` with your professional portrait.

Recommended specs:
- High-resolution portrait orientation (ideally 4:5 or 3:4 ratio)
- Clean, well-lit, confident expression
- Preferably against a simple or elegant background

The site will automatically use this image via the Headshot component.

After replacing, you can run `npm run dev` and the new photo will appear on the About section.
